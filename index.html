Introduction

This Privacy Policy explains how GameRadar ("we", "our", or "the bot") collects, uses, and protects information when you use our Discord bot service. By using GameRadar, you agree to the collection and use of information in accordance with this policy.

Information We Collect

Automatically Collected Information

  • Guild (Server) Information: Server ID, server name, and configuration settings
  • Channel Information: Channel IDs where the bot is configured to send notifications
  • User Commands: Commands executed by users for bot functionality
  • Message Data: Only commands directed to the bot (messages starting with `$`)

User-Provided Information

  • Configuration Settings: Filters, language preferences, and notification preferences set by server administrators
  • Channel Designations: Channels selected for game deal notifications

How We Use Your Information

We use the collected information solely to:

  • Provide game deal notifications and free game alerts
  • Maintain server-specific configuration settings
  • Respond to user commands and provide bot functionality
  • Track previously sent deals to prevent duplicate notifications

Data Storage and Security

Local Storage

  • All data is stored locally in JSON files (`guilds.json` and `seen.json`)
  • No data is transmitted to external servers or third parties
  • Configuration data includes: server preferences, notification channels, and filter settings

Data Security

  • We implement reasonable security measures to protect your data
  • Access to bot data is restricted to authorized bot administrators only
  • No sensitive personal information is collected or stored

Data Sharing and Third Parties

No Data Sales

We do not sell, trade, or rent your information to third parties.

API Usage

The bot fetches public information from:

  • Steam Store API (for game deals)
  • CheapShark API (for multi-store deals)
  • GamePower API (for Epic Games free games)

No personal data is shared with these services.

Discord Integration

As a Discord bot, we operate within Discord's platform and are subject to Discord's Terms of Service and Privacy Policy.

Data Retention

  • Configuration Data: Retained as long as the bot remains in your server
  • Deal History: Maintained to prevent duplicate notifications; cleaned periodically
  • User Commands: Not permanently stored; processed in real-time

Your Rights and Choices

Server Administrators Can:

  • Configure notification settings and filters
  • Set or change notification channels
  • Remove the bot from their server at any time

Data Deletion

When the bot is removed from a server:

  • You can request complete data deletion by contacting the bot developer

Children's Privacy

GameRadar is not directed to children under 13. We do not knowingly collect personal information from children under 13. If you become aware that a child has provided us with personal information, please contact us.

Changes to This Privacy Policy
